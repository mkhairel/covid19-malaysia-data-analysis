# COVID-19 Malaysia Data Analysis

## Project Overview
This project analyzes COVID-19 cases in Malaysia using Python.
It focuses on daily trends, cumulative cases, and monthly patterns, including interactive visualizations.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly

## Key Analysis
- Daily COVID-19 cases trend
- Cumulative cases growth
- Monthly comparison
- Top 10 highest daily cases

## Dataset
Source: Malaysia Ministry of Health (Public COVID-19 Dataset)
